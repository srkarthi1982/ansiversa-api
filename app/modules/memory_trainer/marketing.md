## App Summary

Memory Trainer is an Ansiversa learning app that helps authenticated users create memory games, start practice sessions, submit round answers, and review performance.

The current live V1 supports owner-scoped memory games, server-created practice sessions, generated round sequences, answer submission, round scoring, session performance calculation, review detail, and progress APIs.

It does not include public leaderboards, multiplayer play, adaptive difficulty, shared game templates, scheduled practice, or clinical memory assessment.

## Problems We Solve

Memory practice often lacks structure, scoring, and review history.

Memory Trainer helps by:

- Letting users create memory games
- Starting server-owned practice sessions
- Generating round sequences
- Accepting sequence answers
- Scoring submitted rounds
- Calculating session performance
- Showing review and progress information
