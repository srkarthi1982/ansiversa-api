## App Summary

Lesson Builder is an Ansiversa education app that helps authenticated users create lesson plans, arrange ordered lesson sections, and publish a finished lesson.

The current live V1 supports owner-scoped lesson plans, ordered sections, section reordering, and publishing. Published lessons are locked by service logic so the finished lesson remains stable.

It does not include collaborative editing, attachments, classroom delivery, export formats, templates, versioned publishing, or Course Tracker integration in the current version.

## Core Value Proposition

Lesson Builder gives educators and content creators a focused authoring workflow for turning lesson ideas into ordered lesson plans.

The value is structured lesson creation and stable publishing, not classroom delivery or automated content generation.

## Problems We Solve

Lesson planning can become scattered across notes, documents, and incomplete outlines.

Lesson Builder helps by:

- Saving lesson metadata
- Capturing subject, audience, duration, and objective
- Adding ordered lesson sections
- Supporting section types such as objective, instruction, activity, assessment, and resource
- Reordering sections
- Publishing a stable locked lesson
