## App Summary

Task Prioritizer is an Ansiversa productivity app that helps authenticated users decide which owner-scoped tasks should receive attention first using explicit task signals, local scoring, manual priority overrides, and review history.

The current live V1 supports protected Prioritize, Tasks, History, and Insights routes. Users can create tasks, edit details, duplicate tasks, delete tasks, assign manual priorities, recalculate local priorities, search and filter, review priority history, and inspect dashboard summary signals.

It does not support AI prioritization, recurring tasks, team assignments, comments, attachments, calendar sync, reminders, Kanban boards, import/export, external task providers, cross-app handoffs, or collaboration in the current version.
