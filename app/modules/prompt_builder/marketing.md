## App Summary

Prompt Builder is an Ansiversa AI productivity app that helps authenticated users create, organize, reuse, categorize, and version AI prompt work.

The current live V1 stores prompt projects, editable prompt records, reusable templates, and prompt history events in an owner-scoped isolated backend. It supports prompt organization and workflow history, not prompt execution.

It does not generate prompts with an AI model, execute prompts, score prompts, store execution results, share templates, import/export prompts, or refine prompts automatically in the current version.

## Problems We Solve

Prompts often live in notes, chats, old documents, and copied fragments.

Prompt Builder helps by:

- Organizing prompts by project
- Storing editable prompt text and context
- Saving reusable templates
- Tracking revision, testing, usage, versioning, and archive events
- Returning previews in dashboards and full content only in detail views
- Keeping prompt work owner-scoped
