## App Summary

Grammar and Paraphrasing Assistant is an Ansiversa writing app that helps authenticated users save source text, run lightweight grammar or paraphrase passes, and review generated results.

The current live V1 uses deterministic placeholder generation. It can normalize spacing, capitalization, punctuation, and tone-framed paraphrase text. It is not a real grammar engine, external AI provider integration, or advanced writing assistant in the current version.

## Core Value Proposition

The app gives users a place to organize source text, generate simple V1 results, compare outputs, and revisit writing history.

The value is structured review, not automated perfection.

## Problems We Solve

Writers often paste text into temporary tools, lose earlier versions, and cannot easily review what changed.

This app helps by:

- Saving writing projects
- Preserving source text
- Running lightweight correction or paraphrase actions
- Storing result records
- Keeping job history
- Separating project lists from full text detail views
