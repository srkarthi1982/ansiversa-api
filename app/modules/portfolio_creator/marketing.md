## App Summary

Portfolio Creator is an Ansiversa career app that helps authenticated users structure professional portfolio content through profiles, projects, skills, and publish settings.

The current live V1 stores portfolio profiles, project highlights, skills with categories and proficiency, and publish settings with visibility, slug, theme, and published state. It supports multiple portfolio variants and owner-scoped editing.

It does not render public hosted portfolio pages, manage custom domains, upload media assets, track analytics, import records from other career apps, or create shareable public portfolio URLs in the current version.

Portfolio Creator gives users a persistent place to structure portfolio content before public rendering, media storage, analytics, and shareable publishing features arrive.

## Problems We Solve

Portfolio work often lives across resumes, notes, old links, and scattered project descriptions.

Portfolio Creator helps by:

- Organizing profile content
- Capturing project highlights
- Recording skills by category and proficiency
- Configuring publish settings
- Supporting multiple portfolio variants
- Showing readiness indicators through dashboard counters
