## App Summary

Markdown Editor is an Ansiversa browser-first utility app for writing Markdown, previewing rendered HTML, copying content, and exporting files locally.

The current live V1 keeps runtime editing, preview rendering, copy actions, downloads, clear, reset, draft text, rendered HTML, and stats in the frontend. The backend only provides catalog and overview metadata.

It does not save account-backed documents, version history, collaboration data, uploaded assets, rendered HTML records, or app-specific runtime database records in V1.
