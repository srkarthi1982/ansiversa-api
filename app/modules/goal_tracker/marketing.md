## App Summary

Goal Tracker is an Ansiversa productivity app that helps authenticated users create long-lived goals, break them into milestones, and record dated progress check-ins.

The current live V1 includes owner-scoped goal CRUD, duplication, milestone management, check-ins, dashboard summaries, status and priority signals, and progress review.

It does not include reminders, recurring goals, calendar sync, collaboration, automated progress scoring, AI coaching, exports, or cross-app recommendations in the current version.

## Core Value Proposition

Goal Tracker gives users a structured place to define goals, break them into milestones, and record progress over time.

The value is clarity and accountability, not prediction, automation, or pressure.

## Problems We Solve

People often set goals in one place, track milestones somewhere else, and forget the progress signals that show what is actually happening.

Goal Tracker helps by:

- Keeping goals organized
- Breaking work into milestones
- Supporting dated check-ins
- Recording mood, notes, and progress
- Showing active, paused, completed, and high-priority goals
- Letting users duplicate useful goal structures
