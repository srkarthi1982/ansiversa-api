## App Summary

Project Tracker is an Ansiversa productivity app that helps authenticated users organize user-owned projects and tasks in a persistent planning workspace.

The current live V1 supports protected Projects, Tasks, and Insights routes. Users can create projects, add tasks, update saved records, delete records, and review persisted progress signals from a dashboard endpoint.

It does not include team collaboration, comments, attachments, recurring tasks, calendar sync, templates, time tracking, notifications, Kanban boards, Gantt charts, import/export, AI summaries, or enterprise project-management features in the current version.

## Core Value Proposition

Project Tracker gives users a persistent planning space for projects and tasks without the complexity of collaboration infrastructure or enterprise project-management systems.

The value is personal clarity and progress review, not team workflow automation.

## Problems We Solve

Small projects often become scattered across notes, messages, and incomplete task lists.

Project Tracker helps by:

- Storing project records
- Storing project tasks
- Supporting status, priority, and due-date review
- Separating summary responses from full edit details
- Showing active projects, completed projects, blocked tasks, overdue tasks, and upcoming due dates
- Keeping records owner-scoped
