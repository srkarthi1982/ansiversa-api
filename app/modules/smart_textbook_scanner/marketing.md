## App Summary

Smart Textbook Scanner is an Ansiversa learning app that helps authenticated users organize textbook scan projects, pages, extracted notes, and review metrics.

The current live V1 stores scan structure, page text, extraction status, extracted notes, dashboard summaries, and review aggregates. Users manually create scans, add textbook pages, update page text or status, and create extracted notes from pages.

It does not provide file uploads, OCR processing, image storage, automated note extraction, concept linking, exports, or AI-assisted extraction in the current version.

## Brand Identity

Smart Textbook Scanner belongs to the Ansiversa ecosystem.

Its identity should feel educational, organized, study-focused, careful, practical, and honest about scanning boundaries.

Avoid camera/OCR claims before implementation, automatic extraction claims, AI tutor claims, textbook piracy implications, and shortcuts that replace studying.

## Problems We Solve

Textbook study notes can get scattered across pages, notebooks, screenshots, and summaries.

Smart Textbook Scanner helps by organizing scan projects, page records, extracted notes, extraction statuses, and review metrics in one owner-scoped workflow.
