## App Summary

Password Generator is an Ansiversa browser-first security utility that helps users generate passwords locally after discovering the app through the platform catalog.

The current live V1 keeps password generation, option editing, strength display, copy, and reset actions entirely in the frontend at `/password-generator/generator`. The backend only serves catalog and overview metadata.

It does not persist generated passwords, generation options, secret material, saved history, vault records, breach monitoring, sync, or organization policies in the current version.
