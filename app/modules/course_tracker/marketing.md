## App Summary

Course Tracker is an Ansiversa learning app that helps authenticated users track courses, ordered modules, progress logs, and review metrics. The current live V1 gives users an owner-scoped way to record course structure and learning effort without relying on browser state.

The product supports personal learning accountability. It is not an LMS, gradebook, school administration system, certification authority, course marketplace, or external course provider.

## Brand Identity

Course Tracker belongs to the Ansiversa ecosystem. Every marketing asset should feel simple, trustworthy, intelligent, professional, accessible, and globally understandable.

The brand identity should communicate:

- Simplicity: course progress should be easy to update.
- Trust: self-tracked progress should be clearly distinct from official provider records.
- Intelligence: review metrics should support reflection.
- Professionalism: useful for students, online learners, professionals, and self-learners.
- Accessibility: progress should be understandable without LMS complexity.
- Accountability: the emotional center is steady learning momentum.

Avoid:

- Official certification or grade claims.
- LMS, school administration, or marketplace positioning.
- Scraping or unauthorized provider integration claims.
- AI progress interpretation as current functionality.
- Treating completion metrics as mastery proof.

## Core Value Proposition

Learners take courses across platforms, schools, workplaces, and self-study sources. Course Tracker gives users a personal place to record course details, modules, progress logs, and review metrics so learning progress does not remain trapped in separate systems or memory.

Its strongest value is user-owned accountability. It tracks personal progress without claiming official grades, certificates, or mastery.

## Problems We Solve

- Course progress is scattered across platforms and notes.
- Users need a clear list of active, paused, and completed courses.
- Modules and learning effort need a shared record.
- Study logs lose value when not tied to courses.
- Completion percentages can be misleading without context.
- Users need accountability without school-system complexity.
- External course data should not be scraped or assumed.
