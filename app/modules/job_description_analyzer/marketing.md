## App Summary

Job Description Analyzer is an Ansiversa career app that helps authenticated users turn job postings into structured review records.

The current live V1 stores job descriptions, analysis summaries, extracted skill matches, and analysis history. The backend records analysis data supplied by the frontend.

It does not call an AI model, parse uploaded files, generate automatic match scores, compare resumes, or create authoritative hiring recommendations in the current version.

## Core Value Proposition

The app gives users a structured place to save and review job description analysis data.

The current value is organization and review, not automatic AI interpretation or hiring prediction.

## Problems We Solve

Job postings are dense, inconsistent, and easy to lose after the first read.

Job Description Analyzer helps by:

- Saving job descriptions
- Recording analysis summaries
- Capturing keywords, responsibilities, and recommendations
- Storing skill matches
- Logging review history
- Keeping career planning records owner-scoped
